# מה עושים היום?

A small Hebrew kids activity picker for the browser.

## Play online
This game is hosted with GitHub Pages:
https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/

## What it does
- Lets kids choose a mood/activity type
- Randomly recommends an activity
- Includes a parent mode for managing activities
- Saves activities and categories in the browser with localStorage

## Tech
- Single static `index.html`
- Tailwind via CDN
- Google Fonts
- Plain JavaScript
- No backend required

## Notes
- Data is saved locally in the browser, so it does not sync between devices
- Best used on phone/tablet as well as desktop

## Deploy on GitHub Pages
1. Create a new GitHub repository
2. Upload `index.html`
3. Add this `README.md`
4. Go to **Settings → Pages**
5. Under **Build and deployment**, choose **Deploy from a branch**
6. Select **main** and **/ (root)**
7. Save
8. Open the URL GitHub gives you

## Local development
Just open `index.html` in a browser.
